# Команды git
Инициализация
```sh
git init
```
Различия
```sh
git diff
```
Список коммитов
```sh
git log
```
добавить файл
```sh
git add file_name
```
Коммит
```sh
git commit -m "сообщение"
```
Перемещение по веткам/коммитам
```sh
git checkout 
```
Список веток
```sh
git branch
```
Различия еще раз для конфликта
```sh
git diff
```
Создать новую ветку
```sh
git branch branch_name
```
Слить ветки
```sh
git merge
```
